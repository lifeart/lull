mesh-playback — live demo

Turn old iPhones/iPads into remotely-controlled white-noise speakers — running here entirely in your browser.

This is a real demo with no server. GitHub Pages is static-only, so the hub is simulated in your browser (a leader-elected BroadcastChannel hub running the project's actual shared protocol). The Speaker and Controller below are the unmodified PWAs, genuinely talking to each other. Arm the Speaker, then drive it from the Controller: start/stop, volume, a sleep timer, switch sounds, run the bedtime pre-flight.

🔊 Speaker — the old iPad (arm it once)

→ watch the Controller alarm

🎛 Controller — the parent's phone

Controls every armed room; alarms you when one fails.

← Back to the intro · you can also open the Speaker and the Controller in separate tabs — they sync through the same in-browser hub. Cross-device control needs the real hub (a tiny Node process); see the repository and docs/DESIGN.md for the iOS constraints and how to self-host.

The demo can't reproduce true iOS background/lock behavior (that needs a real device overnight) — it demonstrates the control plane, tiers, timer, and the parent-phone safety alarm.
