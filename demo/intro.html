The honest part

No web app can wake a locked, idle old device from silence at 3am. So the design keeps the sound already playing, and moves failure-detection to your (awake) phone — a loud alarm the moment a room drops. The baby monitor works with the speaker's screen on.

Run it for real

npm install && npm run bake && npm start — then deploy the tiny hub to a Raspberry Pi, a Synology NAS, or behind a Cloudflare Tunnel. HTTPS unlocks the modern-tier extras; plain HTTP on the LAN covers the always-audible core.
